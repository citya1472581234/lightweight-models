# MobileNet

* MobileNet是基於深度分離卷積。深度可分離卷積：把標準卷積分解成深度卷積(depthwise convolution)和逐點卷積(pointwise convolution)，
  這麼做的好處是可以大幅度降低參數量和計算量。
